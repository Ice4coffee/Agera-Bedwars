Официальный бот дискорд сервера Agera Bedwars для верификации игроков и отображения топов BedWars.

🔹 Верификация: по команде !verify игрок получает роль «Верифицирован», а его никнейм меняется на формат [Уровень] Ник.
🔹 Лидерборды: в отдельном канале автоматически обновляются списки лучших игроков по уровню, победам и другим параметрам.
🔹 Надёжность: все конфиденциальные данные хранятся в переменных окружения, код оптимизирован для работы на Render.com.
